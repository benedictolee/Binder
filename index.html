AI 서비스에 접근할 수 있는 나만의 비밀번호예요. 이 앱은 여러분의 Key를 직접 사용해서 서버 비용을 아껴요. Gemini는 무료로 꽤 많이 쓸 수 있어요.

✦ Google Gemini (추천 — 무료)
1
Google AI Studio 접속
Google 계정으로 로그인해요.
aistudio.google.com →
2
Get API Key 클릭
왼쪽 메뉴에서 'Get API key' 버튼을 눌러요.
3
Create API Key → 복사
'Create API key in new project'를 누르고 AIzaSy...로 시작하는 키를 복사해요.
4
앱에 붙여넣기
이전 화면으로 돌아가서 붙여넣으면 완료!
